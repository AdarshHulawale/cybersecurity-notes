## Offensive Security
Offensive security focuses on identifying vulnerabilities in systems by simulating real-world attacks.

Key concepts:
- Penetration testing
- Ethical hacking
- Finding and reporting vulnerabilities before attackers exploit them

Tool:- dirb
DIRB (Directory Buster) is a command-line web content scanner used in web application security testing 
to find hidden or existing files and directories on a web server. It is a valuable tool for ethical hackers 
and security professionals during the reconnaissance phase of a penetration test.
syntax:
dirb website-URL
---

## Defensive Security
Defensive security focuses on protecting systems, networks, and data from attacks.

Key concepts:
- Monitoring and logging
- Incident response
- Threat Intelligence
- Vulnerability Management
- Blue team operations
- Security tools like firewalls and IDS/IPS
- Firewall

---

## Careers in Cyber Security
Cybersecurity offers multiple career paths, including:
- SOC Analyst
- Penetration Tester
- Security Engineer
- Incident Responder
- Threat Intelligence Analyst

Each role focuses on different aspects of protecting or testing systems.

---

## Key Takeaway
Understanding cybersecurity starts with knowing how systems communicate and how attackers and defenders interact. A strong foundation in networking and security concepts is essential for any cybersecurity role.

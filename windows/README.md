# Windows Fundamentals

This folder contains notes from the Windows Fundamentals modules completed on TryHackMe as part of the Pre Security learning path.

Topics covered include:
- Windows file systems (NTFS)
- User and permission management
- Windows services and processes
- System configuration and monitoring tools
- Event logging and system management
- Windows security features (UAC, Firewall, BitLocker, TPM)

These notes focus on understanding Windows internals and security concepts relevant to cybersecurity and SOC roles.
